# 🚀 Asteroid Roll
隕石避けゲーム。  
ロケットを操作して、上から落ちてくる隕石を避け続ける。

---

## 🎮 遊び方

- **ロケットを左右に動かす**
  - キーボード：← →（または A / D）
  - マウス：動かすだけで追従
  - タップ / ドラッグ：指の位置に移動

- **隕石に当たったらゲームオーバー**
- 生き残った時間が **スコア** になります。

---

## 🪨 登場するPNGアセット

- `rocket.png`：プレイヤーのロケット
- `inseki01.png`：隕石（小）
- `inseki02.png`：隕石（大）

---

## 📦 特徴

- 必要最低限のシンプルなルール  
- PNG画像を使った見やすいデザイン  
- 生存時間だけで競える分かりやすいゲーム性  
- マウス・キーボード・タッチに対応
