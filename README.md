### Asteroid Chain Reaction

シンプルなブラウザゲームです。  
隕石を避けながらスコアを稼ぎます。

---

##### normal ver
- 基本的なゲームバージョン
- プレイヤーが隕石に当たるとゲームオーバー
- 隕石は上からランダムに落下
- マウスまたはキーボードで左右移動
- 当たり判定は円ベースで安定化済み
[normal ver](https://zenn24ct.github.io/orbit-shoutotsu-game/normal)

---

##### upgrade ver
- normal ver をベースにした拡張版
- **黄色い球（gold）に当たると一定時間無敵になる**
- 無敵中は隕石に当たってもゲームオーバーにならない
- 視覚エフェクト付き無敵状態
- 今後さらに機能を追加予定
[upgrade ver](https://zenn24ct.github.io/orbit-shoutotsu-game/upgrade)
